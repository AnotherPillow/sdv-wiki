# sdv.wiki

URL shortener for https://stardewvalleywiki.com & https://stardewmodding.wiki.gg

## Redirect Format

- `https://sdv.wiki/m:Index` -> `https://stardewvalleywiki.com/Modding:Index`
- `https://sdv.wiki/Abigail` -> `https://stardewvalleywiki.com/Abigail`
- `https://sdv.wiki/m/Category:Recommendations` -> `https://stardewmodding.wiki.gg/wiki/Category:Recommendations`

## Running

environment variables: `PORT`
